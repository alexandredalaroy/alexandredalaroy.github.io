MorseSpy

Description du projet

MorseSpy est un jeu éducatif qui permet d'apprendre l'alphabet morse de manière ludique. Ce jeu est destiné aux élèves de primaire qui souhaitent apprendre le code morse. Le jeu est composé de plusieurs niveaux de difficulté qui permettent de s'adapter à tous les niveaux. Le but du jeu est de traduire des mots en code morse le plus rapidement possible. Le joueur peut également défier ses camarades en essayant de battre leur score. Ils peuvent donc d'abord commencer à apprendre l'alphabet morse en mode apprentissage, puis s'entraîner en mode jeu pour tester leurs connaissances. Il existe ici plusieurs catégories de mots à traduire, ce qui permet de varier les plaisirs et de ne pas se lasser, allant de simple mots de la vie quotidienne à des mots plus complexes.

Composants clés

  • Importation de données depuis un fichier CSV
  • Sauvegarde de données dans un fichier CSV
  • Affichage dans une console Unix

Technologies utilisées

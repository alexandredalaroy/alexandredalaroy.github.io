CampusTalk

Description du projet

Ce projet a pour but de développer une application Web de messagerie instantanée. L'application permet de discuter en temps réel avec d'autres utilisateurs connectés. Elle est composée de plusieurs salons de discussion, dans lesquels les utilisateurs peuvent échanger des messages. Les utilisateurs peuvent également créer des groupes de discussion et inviter d'autres utilisateurs à les rejoindre. Elle gère la connexion, l'inscription et la déconnexion des utilisateurs. Toutes les données sont stockées dans une base de données (Messages, Utilisateurs, Salons). L'application est développée en Java Enterprise Edition (JEE), des fichiers .jsp, .html et utilise Bootstrap pour le design.

Composants clés

  • Gestion de base de données
  • Utilisation d'un serveur Tomcat
  • Liaison entre des Web Servlet, des JSP et des fichiers HTML

Technologies utilisées

Java Java Enterprise Edition (JEE) JSP Servlet HTML CSS Bootstrap SQL PostgresQL
